# Stumberg Agent

## Setup

1.  **Environment Variables**: Ensure `.env` is populated.
2.  **Dependencies**: `pip install -r requirements.txt`

## Running the App

### Streamlit Interface (Recommended)
To start the web chat interface, run the following command (no compilation required):
```bash
streamlit run app.py
```

### CLI (Legacy)
To run the agent in the terminal:
```bash
python main.py
```
